# Xin chào, tôi là Nguyễn Sinh

Tôi là lập trình viên tập trung vào phát triển phần mềm desktop, hệ thống tự động hóa, quản trị dữ liệu và nền tảng automation.

## Tôi đang xây dựng

- MiniSoftwaresDesktop — nền tảng desktop automation chuyên nghiệp
- TestLab Cloud — hệ thống quản lý task, workspace và module
- ViewCheo Mobile — ứng dụng chạy task trên thiết bị PHONE
- Automation Script Studio — trình thiết kế script/DSL cho tự động hóa

## Kỹ năng chính

- C# / WPF / MVVM
- ASP.NET Core / API Server
- SQLite / SQL Server
- Desktop Application
- Browser Automation
- Data Extraction
- UI/UX Architecture
- Git / GitHub

## Dịch vụ có thể hỗ trợ

- Xây dựng phần mềm desktop theo yêu cầu
- Thiết kế tool automation
- Xây dựng dashboard quản trị
- Thiết kế hệ thống quản lý dữ liệu
- Tối ưu UI/UX cho phần mềm
- Tư vấn kiến trúc phần mềm
