### MiniSoftwares Desktop Automation Platform

A Windows desktop automation platform for managing browser profiles and automating web workflows.

**Technologies:** C#, WPF, .NET 8, Selenium WebDriver, Entity Framework Core, SQLite, MailKit, Win32 API, MVVM

**Main features:**

- Browser profile management
- Account, proxy, cookie, email and OTP management
- Multi-step script execution
- Chrome automation using Selenium WebDriver
- Runtime logging and status tracking
- Local data persistence with SQLite and EF Core

### MiniSoftwares Android Automation Platform

A modular Android automation platform supporting script authoring, DSL-based runtime execution, overlay tools, logging, reports, traces, and artifacts.

**Technologies:** Kotlin, Jetpack Compose, Hilt, Retrofit, Android SDK, Gradle, Modular Architecture

**Main features:**

- Automation DSL with actions such as Tap, Input, Wait, Scroll, Swipe, Screenshot, Dump, Log and Report
- Control flows including If/Else, While, Repeat, ForEach and Try/Catch/Finally
- Runtime execution plan generation
- Logging, reporting and validation
- Jetpack Compose editor UI

### MiniSoftwares Web Admin & API System

A web administration and backend API system for operating MiniSoftwares products.

**Technologies:** PHP, MySQL, HTML, CSS, JavaScript, REST API

**Main features:**

- Payment callback handling

- Release management
